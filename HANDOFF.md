## What Katha is
An AI-powered interactive **visual novel** set in Indian mythology — you don't read the epic, you
live inside it. Every character (Karna, Krishna, Draupadi, Arjuna in Phase 2; Betaal in Phase 1) is
an independent AI agent with persistent memory and an isolated knowledge state (they only know what
they witnessed). Voice-first (Hindi/English via Sarvam). **Not a chatbot — a visual novel with AI
NPCs.** Format: full-screen scene art + character portrait + bottom dialogue panel with typewriter +
voice toggle. *Genshin story-mode / Fate-style, but AI-generative and Indian.*

**Market validation:** Mythik raised ~$20M doing AI Indian-mythology *passive video*; Katha is the
*interactive* layer nobody has built.
