About Katha
Every character is an AI agent that can only retrieve facts it personally witnessed — secrecy enforced in the retrieval layer (the "witness gate"), not by prompting. A character literally cannot reveal what it did not see, however you phrase the question. Proven leak-proof by an automated eval: 0 leaks across a 28-probe adversarial set. Built on a 6-node multi-agent graph + 3-layer RAG, provider-swappable LLM, runs offline on a deterministic mock (optional Gemini for dynamic dialogue).
